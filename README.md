# DATA BRIDGE

DataBridge adalah layanan data ingestion dan transformation yang memungkinkan pengiriman data ke berbagai database dengan struktur yang berbeda melalui sistem mapping dinamis dan pipeline pemrosesan yang scalable.

## Main Features

- Dynamic Schema Mapping
- Multi Database Connector
- Config-Based Transformation
- Validation Layer
- Asynchronous Processing
- Retry & Error Handling
- Plugin-Based Database Adapter

## Supported Databases

- MySQL
- PostgreSQL
